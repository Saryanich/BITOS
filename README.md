## ✨ Ключевые Особенности

### 🚀 1. Интерактивный SplashScreen (Экран загрузки)
* **Кастомный GUI:** Экран загрузки со стильным кастомным градиентом и динамическим рендерингом частиц (эффект звездного неба) на базе `Tkinter.Canvas`.
* **Инициализация систем:** Многопоточный запуск ProgressBar со статус-текстом, отображающим реальные этапы проверки целостности файлов, генерации серийных номеров, проверки оборудования и подготовки графического интерфейса.

### 🔒 2. Безопасность и Авторизация (LoginScreen)
* **Защита от Брутфорса:** Ограничение количества попыток ввода PIN-кода (максимум 5). При превышении лимита подсистема безопасности временно блокирует ввод на 60 секунд.
* **Хеширование данных:** Безопасное хранение пользовательских PIN-кодов с использованием криптографического алгоритма `SHA-256`.
* **Управление питанием:** Интеграция с WinAPI — реальная возможность выключения (`shutdown /s /t 1 /f`) и перезагрузки (`shutdown /r /t 1 /f`) хост-системы Windows прямо из интерфейса эмулятора.
* **Смена PIN-кода:** Встроенное диалоговое окно для быстрой и безопасной смены текущего пароля пользователем.
* **Блокировка клавиатуры (Анти-выход):** При включении экрана блокировки библиотека `keyboard` перехватывает и жестко блокирует системные клавиши Windows (включая `Win`, `Alt+Tab` и др.), предотвращая случайное сворачивание эмулятора и выход в хост-систему.

### 📂 3. Рабочий Стол и Проводник (Explorer)
* **Менеджер Ярлыков:** Автоматическое размещение иконок по сетке, поддержка Drag-and-Drop для перетаскивания элементов, контекстные меню для управления ярлыками.
* **Встроенный Проводник:** Навигация по доступным директориям, создание папок и файлов популярных форматов (`.txt`, `.html`, `.py`, `.js`, `.json`) с возможностью их немедленного редактирования во встроенном текстовом редакторе.
* **Корзина (TrashManager):** Безопасное удаление файлов с автоматическим перемещением в изолированную временную папку, возможностью полного восстановления в исходное место или безвозвратного уничтожения.

### 🔔 4. Системный Трей, Уведомления и Звук
* **Центр уведомлений:** Всплывающие анимированные окна (Toast notification) с историей системных событий, звуковым сопровождением и счетчиком непрочитанных сообщений.
* **Регулятор Громкости:** Кастомный микшер звука с динамической сменой иконок в зависимости от установленного уровня громкости (🔇, 🔈, 🔉, 🔊).

### 📌 5. Кастомизация интерфейса и Виджеты
* **Рабочие Виджеты на Canvas:** Возможность выноса на рабочий стол интерактивных модулей: аналоговых/цифровых часов, календаря и текстовых стикеров-напоминаний с сохранением их координат и текста между перезапусками.
* **Динамические Темы:** Полная поддержка кастомизации интерфейса и изменения цветовых схем (светлая, темная, неоновая) на лету для всех системных окон.

### 🎨 6. Мультимедиа и Приложения
* **Галерея (Галерея изображений):** Просмотр картинок, масштабирование, слайд-шоу и возможность установки изображения в качестве обоев рабочего стола (при наличии Pillow).
* **Браузер (Простой веб-проводник):** Встроенный мини-браузер для рендеринга простых HTML-страниц или выполнения легковесных сетевых запросов (при наличии requests).

### 📝 7. Аудит и Комплексное Логирование (SecurityManager)
Система ведет раздельный и детальный учёт всех действий в сессии для обеспечения максимальной осознанности и безопасности:
* `access.log` — фиксация успешных входов, выходов, ложных попыток авторизации и блокировок интерфейса.
* `files.log` — аудит файловых операций: открытие, создание, редактирование, перемещение в корзину и окончательное удаление.
* `applications.log` — детальная история запуска встроенных приложений, утилит и создания новых ярлыков на рабочем столе.
* `security.log` — фиксация критических событий безопасности, включая брутфорс-атаки, генерацию ключей шифрования и смену PIN-кода.
* `error.log` — трассировка непредвиденных ошибок, исключений и сбоев отдельных модулей операционной системы.

## 💻 Системные Требования

* **ОС:** Windows 7 и новее (требуется для корректной работы системных вызовов управления питанием и WinAPI подсистем).
* **ОЗУ:** от 4 ГБ (для плавной работы анимации интерфейса рекомендуется 8 ГБ).
* **Диск:** 35 МБ свободного места под хранение логов и пользовательских документов.
* **⚠️ ВАЖНОЕ ПРИМЕЧАНИЕ:** Чтобы файловый менеджер заработал полноценно, вам нужно запустить BITOS с внешнего накопителя (флешки). Путь к системному диску `C:\` заблокирован в целях безопасности хост-системы, поэтому просмотр файлов на нем будет недоступен.

## 🛠️ Установка зависимостей

Большая часть базового функционала BITOS работает на стандартной библиотеке Python, однако для поддержки расширенных системных модулей (перехват клавиш, галерея, WinAPI-запросы) необходимо доустановить внешние пакеты.

Для установки выполните следующие команды в вашей консоли:

py -m pip install keyboard
py -m pip install Pillow
py -m pip install pywin32
py -m pip install requests

> **Примечания к библиотекам:**
> * `keyboard` — используется для продвинутой низкоуровневой блокировки горячих клавиш Windows при вызове экрана блокировки.
> * `Pillow` — активирует графическую галерею, просмотрщик и работу с изображениями.
> * `pywin32` — необходим подсистеме монтирования для автоматического детекта подключенных USB-накопителей и флешек.

## ⚙️ Первое включение и Вход

При самом первом запуске BITOS автоматически развернет необходимую инфраструктуру: создаст структуру папок, сгенерирует уникальную лицензию, серийный номер оборудования и конфигурационные файлы безопасности. 

* Первый запуск системы может занять **от 5 до 60 секунд** (в зависимости от скорости диска).
* Все последующие запуски выполняются мгновенно и занимают **от 1 до 15 секунд**.

### Данные для авторизации:
* **🔑 ВХОД:**
  * **Логин:** любой (По умолчанию: `User`)
  * **Пароль:** любой (По умолчанию: `1234`)
* **🔧 РЕЖИМ РАЗРАБОТЧИКА:**
  * **Пароль:** `BIT642Os`

### 🔓 Важная информация по разблокировке клавиатуры!
Если вы запустили систему с установленной библиотекой `keyboard`, она перехватывает и полностью блокирует глобальные клавиши Windows на экране авторизации. 

* **⚠️ ОБРАТИТЕ ВНИМАНИЕ:** Ввод пароля/PIN-кода в поле авторизации **НЕ разблокирует** клавиатуру автоматически. 
* **Порядок разблокировки:** Чтобы принудительно отключить системный перехватчик клавиш и вернуть управление клавиатурой хост-системе Windows, необходимо вручную ввести специальную экстренную комбинацию клавиш: **нажмите 5 раз клавишу `Caps Lock`, а затем сразу 2 раза клавишу `Tab`**. Только после этого блокировка будет полностью снята.

## 💡 Использование

Для запуска эмулятора выполните команду:

python "05V6 (6).pyw"

Для предотвращения появления консольного окна (командной строки) на заднем плане в операционной системе Windows, расширение главного файла установлено как `.pyw`. Это заставляет ОС автоматически запускать интерфейс через скрытый интерпретатор `pythonw.exe`.
